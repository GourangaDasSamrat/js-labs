# Currency Converter


Api URL -> `https://api.exchangerate-api.com/v4/latest/`

## Tech used

- HTML
- JavaScript
- Bootstrap

## Installation guide

1. Install dependence:

```zsh
npm install
```

2. Start live server using python

```zsh
python3 -m http.server 8000
# or if you're in windows
# python3 -m http.server 8000
```

3. Open `localhost:8000` in your browser.

### Requirements (for development)

1. Python 3.x.x
2. Node 20 or higher
3. Modern Browser(like Chrome or Firefox)
